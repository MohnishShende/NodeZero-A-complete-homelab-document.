# qBittorrent

## Image

`lscr.io/linuxserver/qbittorrent`

## Purpose

qBittorrent is the downloader for the media pipeline.

## Notes

- Web UI port: `8080`
- Torrent ports: `6881/tcp`, `6881/udp`
- Persistent path: `/home/nodezero/qbittorrent/config:/config`
- Downloads path: `/mnt/hdd/downloads:/downloads`
